Описание проекта: в проекте реализуются вычисления параметров геометрических фигур, к которым задаются определенные параметры непосредственно в самой программе, например, для прямоугльника вводятся его длина и ширина и после программа выводит его характеристики (параметр, площадь и длину диагонали)
